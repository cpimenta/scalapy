Core (scalapy.core)¶

This module contains the core of scalapy: a set of routines and classes to describe the distribution of MPI processes involved in the computation, and interface with BLACS; and a class which holds a block cyclic distributed matrix for computation.

Routines¶

initmpi(gridshape[, block_shape]) Initialise Scalapack on the current process.
