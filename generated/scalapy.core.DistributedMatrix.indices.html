scalapy.core.DistributedMatrix.indices¶

DistributedMatrix.indices(full=True)¶

The indices of the elements stored in the local matrix.

This can be used to easily build up distributed matrices that depend on their co-ordinates.

Parameters:

full : boolean, optional

If False the matrices of indices are not fleshed out, if True the full matrices are returned. This is like the difference between np.ogrid and np.mgrid.
Returns:

im : tuple of ndarrays

The first element contains the matrix of row indices and the second of column indices.

Notes

As an example a DistributedMatrix defined globally as can be created by:

>>> dm = DistributedMatrix(100, 100)
>>> rows, cols = dm.indices()
>>> dm.local_array[:] = rows + cols
