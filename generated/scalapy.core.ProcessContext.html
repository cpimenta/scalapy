scalapy.core.ProcessContext¶

class scalapy.core.ProcessContext(grid_shape, comm=None)¶

Stores information about an MPI/BLACS process.

Parameters:

gridshape : array_like

A two element list (or other tuple etc), containing the requested shape for the process grid e.g. [nprow, npcol].

comm : mpi4py.MPI.Comm, optional

The MPI communicator to create a BLACS context for. If comm=None, then use MPI.COMM_WORLD instead.

Attributes

grid_shape Process grid shape.
grid_position Process grid position.
mpi_comm MPI Communicator for this ProcessContext.
blacs_context BLACS context handle.
