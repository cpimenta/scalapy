Highlevel Interface (routines)¶

This module presents a high-level interface to ScaLAPACK modelled on the API of scipy.linalg. Presently this is quite limited, and is growing as required.

Routines¶

eigh(A[, B, lower, overwrite_a, ...]) Find the eigen-solution of a symmetric/hermitian matrix.
cholesky(A[, lower, overwrite_a, zero_triangle]) Compute the Cholesky decomposition of a symmetric/hermitian matrix.
dot(A, B[, transA, transB]) Parallel matrix multiplication.
